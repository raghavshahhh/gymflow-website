# Utils Directory

This directory contains utility functions and custom hooks for the GymFlow landing website.

## Structure

- **Hooks**: Custom React hooks (e.g., `useScrollAnimation.ts`, `useBreakpoint.ts`)
- **Helpers**: Utility functions for common tasks
- **Constants**: Shared constants and configuration values

## Naming Convention

- Use camelCase for utility files (e.g., `scrollAnimation.ts`)
- Prefix custom hooks with "use" (e.g., `useBreakpoint.ts`)
- Co-locate tests with utilities (e.g., `useBreakpoint.test.ts`)
