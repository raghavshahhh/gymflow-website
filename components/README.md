# Components Directory

This directory contains reusable UI components for the GymFlow landing website.

## Structure

- **Shared Components**: Reusable components like buttons, cards, etc.
- **Layout Components**: Container, grid, and layout utilities
- **Animation Components**: Scroll-based animation wrappers

## Naming Convention

- Use PascalCase for component files (e.g., `CTAButton.tsx`)
- Co-locate tests with components (e.g., `CTAButton.test.tsx`)
- Use descriptive names that reflect the component's purpose
