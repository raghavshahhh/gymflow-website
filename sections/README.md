# Sections Directory

This directory contains page section components for the GymFlow landing website.

## Structure

Each section represents a major part of the landing page:
- `HeroSection.tsx` - Above-the-fold hero section
- `FeatureOverviewSection.tsx` - Feature cards grid
- `ComparisonSection.tsx` - GymFlow vs generic apps comparison
- `AppScreensSection.tsx` - Screenshot gallery
- `DeepFeaturesSection.tsx` - Detailed feature list
- `TechTrustSection.tsx` - Technology and trust indicators
- `FounderSection.tsx` - Founder note and branding
- `FinalCTASection.tsx` - Final call-to-action
- `Footer.tsx` - Footer with links and copyright

## Naming Convention

- Use PascalCase with "Section" suffix (e.g., `HeroSection.tsx`)
- Co-locate tests with sections (e.g., `HeroSection.test.tsx`)
